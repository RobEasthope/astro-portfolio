Micro:bit is a single page design and development prototype for integration into the rebrand of the BBC's 'Do your bit' Micro:bit challenge.

Built in three days, it is built showcase the design, test how the the front end code would be built into the BBC's custom CMS, and to test performance on the parallax elements of design.
